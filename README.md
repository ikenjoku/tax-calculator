### Installation

Clone repo, install the dependencies and start the app.

```sh
$ git clone https://github.com/ikenjoku/tax-calculator.git
$ cd tax-calculator
$ yarn install
$ yarn start
```

Running tests...

```sh
$ yarn test
```
